# זיהוי ותיוג שירים אוטומטי

תוכנת Desktop מלאה לזיהוי שירים לפי תוכן האודיו (Fingerprint) ועדכון תגיות באופן אוטומטי.

## יכולות עיקריות
- סריקת תיקייה לקבצי אודיו: `mp3`, `flac`, `m4a`, `ogg`, `wav`
- זיהוי שירים דרך AcoustID + MusicBrainz
- השוואה מול YouTube (לצורכי אימות בלבד, ללא הורדה)
- הצגת תוצאות בטבלה עם רמת ביטחון וסטטוס
- אישור/עריכה ידנית לפני שמירה
- גיבוי אוטומטי לקבצים לפני עדכון
- כתיבת תגיות ושינוי שם קובץ לפורמט `Artist - Title.ext`
- רישום כל שינוי בקובץ `changes.log`

## דרישות מקדימות
- Python 3.10 ומעלה
- חיבור אינטרנט פעיל
- Chromaprint (`fpcalc`) מותקן במערכת

## התקנת Python
אם Python לא מותקן, הורד והתקן מאתר Python הרשמי:
- https://www.python.org/downloads/

## התקנת Chromaprint (fpcalc)

### Windows
1. הורד Chromaprint מהגרסאות הרשמיות של הפרויקט.
2. חלץ את הקבצים.
3. הוסף את התיקייה שמכילה `fpcalc.exe` ל-`PATH`.

### macOS
```bash
brew install chromaprint
```

### Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install chromaprint
```

## יצירת API Key חינמי ב-AcoustID
1. היכנס לאתר AcoustID: https://acoustid.org/
2. צור חשבון או התחבר.
3. צור API Key חדש.

## הגדרת API Key בתוכנה
בעת הפעלה ראשונה:
- אם הקובץ `config.json` לא קיים, ייפתח חלון להזנת מפתח AcoustID.
- לאחר הזנה, המפתח יישמר אוטומטית בקובץ `config.json` בתיקיית הפרויקט.

## התקנת תלויות
בתיקיית הפרויקט:
```bash
pip install -r requirements.txt
```

## הפעלת התוכנה
```bash
python main.py
```

## הוראות שימוש
1. לחץ על **בחר תיקייה** (או גרור תיקייה לחלון) ובחר תיקיית שירים.
2. לחץ על **סרוק** כדי להתחיל זיהוי ברקע.
3. בדוק את התוצאות בטבלה:
   - `✅ אושר` — זוהה בביטחון גבוה
   - `⚠️ דורש בדיקה` — זיהוי בינוני
   - `❌ לא זוהה` — זיהוי נמוך/נכשל
4. ערוך פרטים ידנית בפאנל הימני במידת הצורך ולחץ **שמור ידנית**.
5. סמן/בטל שורות לאישור.
6. לחץ **החל שינויים** כדי לבצע גיבוי, כתיבת תגיות ושינוי שם קובץ.

## ספי ביטחון
- `>= 0.85` → `✅ אושר`
- `0.5 - 0.84` → `⚠️ דורש בדיקה`
- `< 0.5` → `❌ לא זוהה`

## פתרון בעיות נפוצות
- **שגיאה על fpcalc**: ודא ש-Chromaprint מותקן ושהפקודה `fpcalc` זמינה ב-`PATH`.
- **שגיאת רשת**: בדוק חיבור אינטרנט.
- **לא נמצאו התאמות**: ייתכן שהקובץ פגום או שהקטע קצר מדי.
- **אין הרשאת כתיבה**: הרץ עם הרשאות מתאימות או שנה הרשאות תיקייה.

## קבצי מערכת חשובים
- `config.json` — מפתח API
- `changes.log` — לוג פעולות
- `.backup/` — גיבויים לקבצים לפני עדכון
